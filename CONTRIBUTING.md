# Contributing to worldcupinvitations

Bug reports and ideas are welcome — open an issue, or use the feedback link in
the app.

## Before you open a pull request

This project is released under the [GNU AGPL v3.0 or later](./LICENSE) and is
**also offered under a separate commercial licence** on request. Offering both
only works while one person holds the copyright in the whole codebase.

So any patch that is more than trivial needs a copyright assignment: by opening a
pull request you agree to assign copyright in your contribution to Ben Richardson,
who will license it back to you and to everyone else under the AGPL. If you would
rather not do that, please open an issue describing the change instead — that is
genuinely useful and carries no such condition.

Please say in the pull request whether any of the contribution was generated by
an AI coding assistant, and confirm you have the right to contribute it.

## Third-party code

Do not paste code in from another project unless its licence is compatible with
the AGPL and you add it to [THIRD-PARTY-NOTICES.md](./THIRD-PARTY-NOTICES.md).
